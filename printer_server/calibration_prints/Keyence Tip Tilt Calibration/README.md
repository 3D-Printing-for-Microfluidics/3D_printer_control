The Keyence Tip/Tilt Calibration is an automated, non-printing routine used to establish the baseline focal alignment of the projection optics. By utilizing a Keyence CL3000 confocal distance sensor, the system measures the distance between the projection optic and the print plane across the XY stage. These measurements allow the printer to automatically adjust the tip and tilt of the optics to achieve a theoretically optimal focus across the entire build area before any resin is cured.

On the OS1 series, it is important to note that this is an indirect measurement of focus. Because the system scans the optics relative to a fixed sensor rather than measuring the active print surface directly, this routine serves as a high-precision starting point rather than a final calibration. Consequently, users must still run the physical "Tip/Tilt Focus Sweep Print" to identify specific offsets and perform final manual adjustments. Furthermore, because the measurement is relative to the stage position, significant X/Y offsets in your print files may result in a localized focus shift that requires further compensation.
