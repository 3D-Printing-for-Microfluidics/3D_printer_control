The Tip/Tilt Focus Sweep Print is a multi-region calibration pattern designed to determine optimal focal distance while identifying angular offsets across the build area. The print features five measurement regions—one in the center and four in the +/- Tip and +/- Tilt directions—with the +/+ and -/- corners physically marked for orientation. For the OS1 series printer, the tip axis maps to the long axis of the print. Each region consists of a 5×5 grid of 25 squares representing focal offsets ranging from −12 to +12 times the defined step size, where the top-left corner of each grid represents the negative extreme and the bottom-right represents the positive extreme.

To calibrate the system, inspect each of the five regions under magnification to identify the "best square" exhibiting the highest clarity and sharpest feature resolution. The center region functions as a compressed focus print to establish the primary focal baseline, while the peripheral regions are used to calculate optical misalignment. Determine the Tip and Tilt offsets by recording the numerical step value of the best square in each region and calculating the difference between opposing axes using the formula: Axis Offset=(Best Pos Square)−(Best Neg Square). Adjust above the tip/tilt offsets by these numbers.
