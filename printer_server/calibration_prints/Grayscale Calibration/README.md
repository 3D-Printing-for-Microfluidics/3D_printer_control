The Grayscale Calibration is a non-printing diagnostic routine that utilizes an onboard photodiode to measure the Digital Micromirror Device (DMD) and generate high-precision correction maps. Unlike standard focus prints, this process performs an internal optical scan to capture light intensity data across the projection field, compensating for optical non-uniformities to ensure consistent exposure and grayscale accuracy. This routine must be successfully completed before grayscale correction can be enabled or utilized during active printing.

The routine systematically addresses variations in light output to create a normalized intensity profile for the optics. By analyzing photodiode feedback, the system generates a corrected grayscale map that adjusts mirror duty cycles to flatten the intensity curve across the entire build area. This calibration is essential for maintaining print uniformity and dimensional accuracy, as it ensures that a specific grayscale value delivers a uniform light dosage regardless of its position on the DMD.
